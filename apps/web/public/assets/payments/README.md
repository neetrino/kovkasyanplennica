# Payment Logos

This directory contains logos for payment providers.

## Required Logos

Please add the following logo files:

1. **idram.svg** - Idram payment logo
2. **arca.svg** - ArCa payment logo
3. **armenian-express.svg** - Armenian Express payment logo
4. **ameria.svg** - Ameria payment logo

## Logo Specifications

- Format: SVG (preferred) or PNG
- Recommended size: 200x60px or similar aspect ratio
- Background: Transparent or white
- Style: Horizontal logo format

## Where to Find Logos

- Official payment provider websites
- Brand guidelines from payment providers
- Contact payment provider support for official logos

## Note

If a logo file is missing, the payment method will still be displayed with text only.
