# Favorites Section Images

This directory should contain the salad images for the "НАШИ ФАВОРИТЫ" (Our Favorites) section.

## Required Images

- `salad-image.png` - Circular image of salad with salmon and fresh greens (used for all 8 product cards)

## Image Specifications

- Format: PNG
- Shape: Circular (will be displayed in a circle)
- Recommended size: 256x256px or larger
- Content: Salad with salmon, fresh greens, and other ingredients

## How to Add Images

1. Download the salad image from Figma design
2. Save it as `salad-image.png` in this directory
3. The image will be automatically used in all 8 product cards
